## 查询产品信息：ATI

语法规则：

| 命令类型 | 语法 | 返回                 |
| -------- | ---- | -------------------- |
| 执行命令 | ATI  | <module info> <br>OK |

 

参数定义：

| 参数          | 定义                       | 取值 | 对取值的说明 |
| ------------- | -------------------------- | ---- | ------------ |
| <module info> | 模块相关信息（厂商，版本） | -    | 由厂商定义   |

 

举例：

| 命令（→）/返回（←） | 实例                             | 解释和说明   |
| ------------------- | -------------------------------- | ------------ |
| →                   | ATI                              | 查询模块信息 |
| ←                   | AirM2M_780EP_V1007_LTE_AT <br>OK | 返回查询结果 |
