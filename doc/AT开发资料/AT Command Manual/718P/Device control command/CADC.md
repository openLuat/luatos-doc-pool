## 读取ADC：AT+CADC

语法规则：

| 命令类型 | 语法                                       | 返回                         |
| -------- | ------------------------------------------ | ---------------------------- |
| 设置命令 | AT+CADC=<adc_id>,<mode>                    | OK                           |
| 读取命令 | AT+CADC=<adc_id>,2                         | +CADC:<adc_id>,<volt> <br>OK |
| 注意     | 读取之前需先打开对应通道，默认最大量程1.2V |                              |

 

参数定义：

| 参数     | 定义             | 取值 | 对取值的说明                                    |
| -------- | ---------------- | ---- | ----------------------------------------------- |
| <adc_id> | ADC id，共有两路 | 0    |                                                 |
|          |                  | 1    |                                                 |
| <mode>   | ADC使用模式      | 0    | 禁用                                            |
|          |                  | 1    | 启用                                            |
|          |                  | 2    | 查询该路adc的电压(EC618系列模块>=V1115版本支持) |
| <volt>   | ADC电压          |      | 单位：毫伏                                      |

 

举例:

| 命令(→)/返回(←) | 实例                 | 解释和说明 |
| --------------- | -------------------- | ---------- |
| →               | AT+CADC=0,1          | 打开ADC0   |
| ←               | OK                   |            |
| →               | AT+CADC=0,2          | 读取ADC0值 |
| ←               | +CADC: 0,1086 <br>OK |            |
