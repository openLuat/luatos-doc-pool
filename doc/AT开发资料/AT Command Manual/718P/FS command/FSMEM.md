## 获取可用空间大小：AT+FSMEM

执行命令用来获取文件系统剩余空间大小。

语法规则：

| 命令类型 | 语法       | 返回                  |
| -------- | ---------- | --------------------- |
| 执行命令 | AT+FSMEM   | +FSMEM: <size> <br>OK |
| 查询命令 | AT+FSMEM=? | OK<br>或<br>ERROR     |

 

参数定义：

| 参数   | 定义                 | 取值 | 对取值的说明 |
| ------ | -------------------- | ---- | ------------ |
| <size> | 文件系统可用空间大小 |      | 单位：字节   |
