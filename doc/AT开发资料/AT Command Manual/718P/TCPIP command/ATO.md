## 从命令模式切换至数据模式：ATO

**注：本命令合宙4G CAT1 EC716S系列平台模块不支持（Air780EL系列& Air780ET系列& Air700EC系列）**

 

语法规则：

| 命令类型 | 语法 | 返回和说明                                                   |
| -------- | ---- | ------------------------------------------------------------ |
| 执行命令 | ATO  | 成功，则返回:<br>CONNECT <br>说明：TA 继续数据传输连接，即从命令模式切换回数据模式 |
|          |      | 若无法恢复到数据模式，则返回：<br>NO CARRIER                 |

 

举例：

| 命令（→）/返回（←） | 实例    | 解释和说明     |
| ------------------- | ------- | -------------- |
| →                   | ATD*99# | 数据拨号       |
| ←                   | CONNECT | 成功           |
| →                   | +++     | 切到AT命令模式 |
| ←                   | OK      | 成功           |
| →                   | ATO     | 再切到数据模式 |
| ←                   | CONNECT | 成功           |
