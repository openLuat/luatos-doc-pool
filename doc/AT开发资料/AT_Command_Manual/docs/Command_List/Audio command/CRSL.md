## 本地音频播放音量控制：AT+CRSL

本命令影响设备本地音频播放的音量。

语法规则：

| 命令类型 | 语法            | 返回                            |
| -------- | --------------- | ------------------------------- |
| 设置命令 | AT+CRSL=<level> | OK                              |
| 查询命令 | AT+CRSL?        | +CRSL:<level><br> OK            |
| 测试命令 | AT+CRSL=?       | +CRSL: (<level>取值范围) <br>OK |

 

参数定义：

| 参数    | 定义     | 取值 | 对取值的说明                      |
| ------- | -------- | ---- | --------------------------------- |
| <level> | 音频通道 | 1~15 | 1声音最小，15声音最大。缺省值为15 |
