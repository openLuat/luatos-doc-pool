## 保存RI设置功能：AT+CFGRISAVE

设置是否保存AT+CFGRI指令设置的值。

语法规则：

| 命令类型 | 语法                                                    | 返回                       |
| -------- | ------------------------------------------------------- | -------------------------- |
| 设置命令 | `AT+CFGRISAVE=<save>`                                   | OK                         |
| 查询命令 | AT+CFGRISAVE?                                           | `+CFGRISAVE:<save>` <br>OK |
| 注意事项 | AT+CFGRI设置对应值后,需执行AT+CFGRISAVE=1才会保持对应值 |                            |

 

参数定义：

| 参数     | 定义         | 取值 | 对取值的说明 |
| -------- | ------------ | ---- | ------------ |
| `<save>` | 是否关键保存 | 0    | 不保存       |
|          |              | 1    | 保存         |
