## 读取VBAT电压：AT+CBC

语法规则：

| 命令类型 | 语法                                | 返回                                     |
| -------- | ----------------------------------- | ---------------------------------------- |
| 读取命令 | AT+CBC                              | `+CBC: <volt>` <br>OK                    |
| 测试命令 | AT+CBC=?                            | OK                                       |
| 设置命令 | `AT+CBC=<powerOnVol>,<powerOffVol>` | OK                                       |
| 读取命令 | AT+CBC?                             | `+CBC:<powerOnVol>,<powerOffVol> `<br>OK |

 

参数定义：

| 参数            | 定义             | 取值 | 对取值的说明 |
| --------------- | ---------------- | ---- | ------------ |
| `<volt>`        | VBAT当前电压     |      | 单位：毫伏   |
| `<powerOnVol>`  | 开机检测的电压值 | >=0  | 单位：毫伏   |
| `<powerOffVol>` | 关机检测的电压值 | >=0  | 单位：毫伏   |
