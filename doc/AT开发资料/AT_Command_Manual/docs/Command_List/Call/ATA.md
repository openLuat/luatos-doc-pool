## 接听来电：ATA

模块有来电(RING)时，输入该命令接听来电

语法规则：

| 命令类型 | 语法 | 返回                                     |
| -------- | ---- | ---------------------------------------- |
| 执行命令 | ATA  | 命令成功，则返回：<br>OK                 |
|          |      | 当前没有建立通话，则返回：<br>NO CARRIER |

 

举例：

| 命令（→）/返回（←） | 实例 | 解释和说明                                                   |
| ------------------- | ---- | ------------------------------------------------------------ |
| <-（来电URC）       | RING | 有RING消息上报，表示有来电。<br>注：RING是一个URC命令（Unsolicited Result Code） |
| ->                  | ATA  | 模块接听来电                                                 |
| <-                  | OK   | 返回OK，说明接听来电成功                                     |
