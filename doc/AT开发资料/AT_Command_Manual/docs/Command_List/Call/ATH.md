## 挂断通话：ATH

通话过程中，输入执行命令会挂断所有电话，包括PPP拨号通道和当前通话（active）、等待通话（waiting）和挂起通话（held）

语法规则：

| 命令类型 | 语法 | 返回                     |
| -------- | ---- | ------------------------ |
| 执行命令 | ATH  | 命令成功，则返回：<br>OK |

 

举例：

| 命令（→）/返回（←） | 实例 | 解释和说明                                                   |
| ------------------- | ---- | ------------------------------------------------------------ |
| <-（来电URC）       | RING | 有RING消息上报，表示有来电。<br>注：RING是一个URC命令（Unsolicited Result Code） |
| ->                  | ATA  | 模块接听来电                                                 |
| <-                  | OK   | 返回OK，说明接听来电成功                                     |
| ->                  | ATH  | 挂断通话                                                     |
| <-                  | OK   | 返回OK                                                       |
