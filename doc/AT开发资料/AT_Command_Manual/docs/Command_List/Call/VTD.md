## DTMF TONES周期：AT+VTD

语法规则：

| 命令类型 | 语法       | 返回                       |
| -------- | ---------- | -------------------------- |
| 设置命令 | AT+VTD=<n> | OK                         |
| 查询命令 | AT+VTD?    | +VTD: <n>  默认值:3 <br>OK |
| 测试指令 | AT+VTD=?   | +VTD:(1-10) <br>OK         |

 

参数定义：

| 参数 | 定义     | 取值 | 对取值的说明                     |
| ---- | -------- | ---- | -------------------------------- |
| <n>  | 持续时间 | 1~10 | tone的持续时间，以1/10秒为单位。 |
