## 关闭TCP连接：AT+MIPCLOSE

语法规则：

| 命令类型 | 语法          | 返回  | 说明                   |
| -------- | ------------- | ----- | ---------------------- |
| 设置命令 | AT+MIPCLOSE   | OK    | 成功                   |
|          |               | ERROR | 失败                   |
| 测试命令 | AT+MIPCLOSE=? | OK    | 返回OK，表示支持此命令 |
