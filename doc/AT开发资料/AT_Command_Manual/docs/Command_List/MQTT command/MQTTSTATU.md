## 查询MQTT连接状态：AT+MQTTSTATU

语法规则：

| 命令类型 | 语法           | 返回                       | 说明                   |
| -------- | -------------- | -------------------------- | ---------------------- |
| 执行命令 | AT+MQTTSTATU   | +MQTTSTATU :<state> <br>OK | 成功                   |
|          |                | ERROR                      | 失败                   |
| 测试命令 | AT+MQTTSTATU=? | OK                         | 返回OK，表示支持此命令 |

 

参数定义：

| 参数    | 定义         | 取值 | 对取值的说明                   |
| ------- | ------------ | ---- | ------------------------------ |
| <state> | MQTT连接状态 | 0    | 离线                           |
|         |              | 1    | 已经登陆认证过，可以PUB数据了  |
|         |              | 2    | 还没认证，需要发送MCONNECT命令 |
