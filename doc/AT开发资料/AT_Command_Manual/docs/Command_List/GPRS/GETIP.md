## 通过cid查询本地IP地址：AT*GETIP

语法规则：

| 命令类型 | 语法             | 返回                            |
| -------- | ---------------- | ------------------------------- |
| 设置命令 | `AT*GETIP=<cid>` | `+GETIP: <ip>,<gateway>` <br>OK |

 

参数定义：

| 参数    | 定义       | 取值 | 对取值的说明  |
| ------- | ---------- | ---- | ------------- |
| `<cid>` | 承载场景id | 1    | PDP上下文标识 |
