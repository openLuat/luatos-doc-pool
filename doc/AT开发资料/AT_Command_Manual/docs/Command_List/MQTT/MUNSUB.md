## 取消订阅主题：AT+MUNSUB

本命令从客户端到服务器，用于取消订阅主题。

 

语法规则：

| 命令类型 | 语法              | 返回                   | 说明 |
| -------- | ----------------- | ---------------------- | ---- |
| 设置命令 | AT+MUNSUB=<topic> | OK <br>UNSUBACK        | 成功 |
|          |                   | ERROR                  | 失败 |
| 测试命令 | AT+MUNSUB=?       | +MUNSUB:<topic><br> OK | 成功 |

参数定义：

| 参数    | 定义     | 取值   | 对取值的说明                              |
| ------- | -------- | ------ | ----------------------------------------- |
| <topic> | 消息主题 | 字符串 | 最大256个字节。可以用""括住，也可以不用"" |
