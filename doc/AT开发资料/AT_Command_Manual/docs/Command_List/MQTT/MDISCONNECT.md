## 关闭MQTT连接：AT+MDISCONNECT

语法规则：

| 命令类型 | 语法             | 返回  | 说明                   |
| -------- | ---------------- | ----- | ---------------------- |
| 设置命令 | AT+MDISCONNECT   | OK    | 成功                   |
|          |                  | ERROR | 失败                   |
| 测试命令 | AT+MDISCONNECT=? | OK    | 返回OK，表示支持此命令 |
