## 查询卡的MCC以及MNC：AT+MNCC

**注：本命令EC618平台系列模块（Air780E系列）软件版本>=V1153版本支持**

语法规则：

| 命令类型 | 语法    | 返回                         |
| -------- | ------- | ---------------------------- |
| 执行命令 | AT+MNCC | `mcc:<mcc>,mnc:<mnc>` <br>OK |

 

参数定义:

| 参数    | 定义    | 取值 | 对取值的说明 |
| ------- | ------- | ---- | ------------ |
| `<mcc>` | 卡的mcc |      |              |
| `<mnc>` | 卡的mnc |      |              |
