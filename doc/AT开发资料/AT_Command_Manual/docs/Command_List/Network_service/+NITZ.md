## URC)NITZ自动上报：+NITZ:<time>,<ds>

语法规则：

| URC                 |
| ------------------- |
| `+NITZ:<time>,<ds>` |

 

参数定义:

| 参数     | 定义   | 取值                 | 对取值的说明                                       |
| -------- | ------ | -------------------- | -------------------------------------------------- |
| `<time>` | 时间   | yy/mm/dd,hh:mm:ss±tz | tz :时区, -48~+48举例:+NITZ:11/08/02,09:27:39+32,0 |
| `<ds>`   | 夏令时 | 0                    | 没有调整夏令时                                     |
|          |        | 1                    | +1小时(等于4个quarter in `<tz>`)调整为夏令时       |
|          |        | 2                    | +2小时(等于8个季度`<tz>`)调整为夏令时              |
