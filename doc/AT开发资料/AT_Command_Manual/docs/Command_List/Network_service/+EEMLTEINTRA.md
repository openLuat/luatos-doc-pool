## LTE工程模式下同频信息：+EEMLTEINTRA

展示LTE工程模式下同频（intra frequency）信息。

 

语法规则：

| URC                                                          |
| ------------------------------------------------------------ |
| `+EEMLTEINTRA: <p1>,<p2>,<p3>,<p4>,<p5>,<p6>,<p7>,<p8>,<p9>` |

 

参数定义：

| 参数   | 解释                       |
| ------ | -------------------------- |
| `<p1>` | index of ENGMODE INTRAFREQ |
| `<p2>` | phyCellId                  |
| `<p3>` | euArfcn                    |
| `<p4>` | rsrp                       |
| `<p5>` | rsrq                       |
| `<p6>` | mcc                        |
| `<p7>` | mnc                        |
| `<p8>` | tac                        |
| `<p9>` | cellId                     |
