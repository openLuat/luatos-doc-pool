## 设置结果码抑制模式：ATQ

此命令用于设置TA 是否向TE 发送结果码。并且此设置不会影响响应内的内容。

语法规则：

| 命令类型     | 语法                                                  | 返回和说明 |
| ------------ | ----------------------------------------------------- | ---------- |
| 执行命令     | `ATQ<n>`                                              | OK         |
| 参数存储模式 | 设置命令的参数可以通过AT&W命令保存到NVM中，重启不丢失 |            |

 

参数定义：

| 参数  | 定义           | 取值 | 对取值的说明         |
| ----- | -------------- | ---- | -------------------- |
| `<n>` | 是否显示结果码 | 0    | 向TE发送结果码       |
|       |                | 1    | 结果码被抑制，不发送 |

 

举例：

| 命令（→）/  返回（←） | 实例       | 解释和说明             |
| --------------------- | ---------- | ---------------------- |
| →                     | ATQ1       | 设置为不返回结果码     |
| ←                     |            | 此时没出现OK这个结果码 |
| →                     | AT+CREG?   |                        |
| ←                     | +CREG: 0,1 | 此时没出现OK这个结果码 |
