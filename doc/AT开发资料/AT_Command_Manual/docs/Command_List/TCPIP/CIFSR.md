## 查询本地IP地址：AT+CIFSR

语法规则：

| 命令类型 | 语法                                                         | 返回和说明     |
| -------- | ------------------------------------------------------------ | -------------- |
| 执行命令 | AT+CIFSR                                                     | `<IP address>` |
| 测试命令 | AT+CIFSR=?                                                   | OK             |
| 注意事项 | 只 有 在 移 动 场 景 已 激 活 的 状 态 ： IP GPRSACT 、 TCP/UDPCONNECTING 、 CONNECT OK 、 IP CLOSE 下 才 可 以 通 过AT+CIFSR 查询得到本地 IP 地址，否则返回 ERROR |                |

 

参数定义：

| 参数           | 定义    | 取值 | 对取值的说明                 |
| -------------- | ------- | ---- | ---------------------------- |
| `<IP address>` | IP 地址 | -    | 字符串参数(字符串需要加引号) |
