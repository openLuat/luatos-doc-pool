## 查询模块FIRMWARE版本：AT+VER

语法规则：

| 命令类型 | 语法              | 返回                    |
| -------- | ----------------- | ----------------------- |
| 执行命令 | AT+VER            | `<firmware ver> `<br>OK |
| 注意事项 | 返回结果与ATI一致 |                         |

 

参数定义：

| 参数              | 定义             | 取值 | 对取值的说明 |
| ----------------- | ---------------- | ---- | ------------ |
| `< firmware ver>` | 模块内部软件版本 | -    | 由厂商定义   |

 

举例：

| 命令（→）/返回（←） | 实例                             | 解释和说明   |
| ------------------- | -------------------------------- | ------------ |
| →                   | AT+VER                           | 查询模块信息 |
| ←                   | AirM2M_780EP_V1007_LTE_AT <br>OK | 返回查询结果 |
