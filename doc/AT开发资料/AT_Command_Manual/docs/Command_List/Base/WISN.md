## 写SN号命令：AT+WISN

AT+WISN是用来写用户 sn(serial number)到模块里的。

语法规则：

| 命令类型            | 语法                | 返回               |
| ------------------- | ------------------- | ------------------ |
| 设置命令            | `AT+WISN=<user_sn>` | OK                 |
| `+CME ERROR: <err>` |                     |                    |
| 查询命令            | AT+WISN?            | `<user_sn>`<br> OK |

 

参数定义：

| 参数        | 定义       | 取值 | 对取值的说明         |
| ----------- | ---------- | ---- | -------------------- |
| `<user_sn>` | special SN | -    | 由用户定义，最长64位 |

 

举例：

| 命令（→）/返回（←） | 实例                 | 解释和说明   |
| ------------------- | -------------------- | ------------ |
| →                   | AT+WISN="1234567890" | 写客户SN号   |
| ←                   | OK                   |              |
| →                   | AT+WISN?             | 查询客户SN号 |
| ←                   | 1234567890 <br>OK    |              |
