## 查询IMSI：AT+CIMI

语法规则：

| 命令类型 | 语法    | 返回和说明      |
| -------- | ------- | --------------- |
| 执行命令 | AT+CIMI | `<IMSI><br> OK` |

 

参数定义：

| 参数     | 定义                                                         | 取值 | 对取值的说明   |
| -------- | ------------------------------------------------------------ | ---- | -------------- |
| `<IMSI>` | 国际移动台用户识别码（International Mobile Subscriber Identity） |      | 由15位数字组成 |

 

举例：

| 命令（→）/返回（←） | 实例                   | 解释和说明   |
| ------------------- | ---------------------- | ------------ |
| →                   | AT+CIMI                | 查询IMSI号   |
| ←                   | 460001841426414<br> OK | 返回查询结果 |
