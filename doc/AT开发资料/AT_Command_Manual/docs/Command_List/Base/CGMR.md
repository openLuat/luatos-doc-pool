## 查询模块版本信息：AT+CGMR

语法规则：

| 命令类型 | 语法      | 返回                          |
| -------- | --------- | ----------------------------- |
| 执行命令 | AT+CGMR   | `Revision: <revision>`<br> OK |
| 测试命令 | AT+CGMR=? | OK                            |

 

参数定义：

| 参数         | 定义         | 取值 | 对取值的说明 |
| ------------ | ------------ | ---- | ------------ |
| `<revision>` | 软件版本标识 |      | 由厂商定义   |

 

举例：

| 命令（→）/返回（←） | 实例                                      | 解释和说明       |
| ------------------- | ----------------------------------------- | ---------------- |
| →                   | AT+CGMR                                   | 查询软件版本标识 |
| ←                   | +CGMR: "AirM2M_780EP_V1007_LTE_AT" <br>OK | 返回查询结果     |
