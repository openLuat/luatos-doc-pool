## 设置FTP数据传输类型：AT+FTPTYPE

语法规则：

| 命令类型 | 语法               | 返回                    | 说明     |
| -------- | ------------------ | ----------------------- | -------- |
| 设置命令 | AT+FTPTYPE=<value> | OK                      | 正常返回 |
| ERROR    | 输入格式有误       |                         |          |
| 查询命令 | AT+FTPTYPE?        | +FTPTYPE:<value> <br>OK |          |
| 测试命令 | AT+FTPTYPE=?       | OK                      |          |

 

参数定义：

| 参数    | 定义            | 取值 | 对取值的说明     |
| ------- | --------------- | ---- | ---------------- |
| <value> | FTP数据传输类型 | "A"  | FTP ASCII字符集  |
|         |                 | "I"  | FTP Binary字符集 |
