## 设置FTP用户名称：AT+FTPUN

语法规则：

| 命令类型 | 语法             | 返回                  | 说明     |
| -------- | ---------------- | --------------------- | -------- |
| 设置命令 | AT+FTPUN=<value> | OK                    | 正常返回 |
| ERROR    | 输入格式有误     |                       |          |
| 查询命令 | AT+FTPUN?        | +FTPUN:<value><br> OK |          |
| 测试命令 | AT+FTPUN=?       | OK                    |          |

 

参数定义：

| 参数    | 定义      | 取值 | 对取值的说明                  |
| ------- | --------- | ---- | ----------------------------- |
| <value> | FTP用户名 |      | 为长度不超过49的ASCII字符串。 |
