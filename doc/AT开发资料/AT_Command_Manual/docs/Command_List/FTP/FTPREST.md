## 设置FTP下载续传：AT+FTPREST

语法规则：

| 命令类型 | 语法               | 返回                    | 说明     |
| -------- | ------------------ | ----------------------- | -------- |
| 设置命令 | AT+FTPREST=<value> | OK                      | 正常返回 |
| ERROR    | 输入格式有误       |                         |          |
| 查询命令 | AT+FTPREST?        | +FTPREST:<value> <br>OK |          |
| 测试命令 | AT+FTPREST=?       | OK                      |          |

 

参数定义：

| 参数    | 定义         | 取值 | 对取值的说明 |
| ------- | ------------ | ---- | ------------ |
| <value> | 要续传的断点 |      |              |
