## 设置FTP输入类型：AT+FTPPUTOPT

语法规则：

| 命令类型 | 语法                   | 返回                        | 说明         |
| -------- | ---------------------- | --------------------------- | ------------ |
| 设置命令 | `AT+FTPPUTOPT=<value>` | OK                          | 正常返回     |
|          |                        | ERROR                       | 输入格式有误 |
| 查询命令 | AT+FTPPUTOPT?          | `+FTPPUTOPT:<value>` <br>OK |              |
| 测试命令 | AT+FTPPUTOPT=?         | OK                          |              |

 

参数定义：

| 参数      | 定义            | 取值   | 对取值的说明   |
| --------- | --------------- | ------ | -------------- |
| `<value>` | FTP数据传输类型 | "APPE" | 添加文件       |
|           |                 | "STOU" | 存储唯一的文件 |
|           |                 | "STOR" | 存储文件       |
