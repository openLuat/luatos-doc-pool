## 获取FTP状态：AT+FTPSTATE

语法规则：

| 命令类型 | 语法          | 返回                      |
| -------- | ------------- | ------------------------- |
| 执行命令 | AT+FTPSTATE   | +FTPSTATE: <state> <br>OK |
| 测试命令 | AT+FTPSTATE=? | OK                        |

 

参数定义：

| 参数    | 定义     | 取值 | 对取值的说明                               |
| ------- | -------- | ---- | ------------------------------------------ |
| <state> | 工作状态 | 0    | Idle                                       |
|         |          | 1    | FTP会话进行中，包括FTPGET、FTPPUT、FTPDELE |
