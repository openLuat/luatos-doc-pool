## 删除EPO文件：AT+CGNSDEL

语法规则：

| 命令类型 | 语法                | 返回                          |
| -------- | ------------------- | ----------------------------- |
| 设置命令 | `AT+CGNSDEL=<mode>` | `+CGNSDEL: <mode>,0,0` <br>OK |
|          |                     | `+CME ERROR:<err>`            |

 

参数定义：

| 参数     | 定义 | 取值 | 对取值的说明 |
| -------- | ---- | ---- | ------------ |
| `<mode>` | 模式 | 1    | 保留EPO文件  |
|          |      | 2    | 保留EPO文件  |
|          |      | 3    | 删除EPO文件  |
