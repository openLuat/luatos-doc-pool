## GPS开关：AT+CGNSPWR

 

语法规则：

| 命令类型 | 语法                  | 返回                        | 说明     |
| -------- | --------------------- | --------------------------- | -------- |
| 设置命令 | `AT+CGNSPWR=<status>` | OK                          | 开关正常 |
|          |                       | ERROR                       | 开关失败 |
| 查询命令 | AT+CGNSPWR?           | `+CGNSPWR:  <status>`<br>OK |          |

 

参数定义：

| 参数       | 定义     | 取值 | 对取值的说明 |
| ---------- | -------- | ---- | ------------ |
| `<status>` | 开关状态 | 0    | 关闭         |
|            |          | 1    | 打开         |
