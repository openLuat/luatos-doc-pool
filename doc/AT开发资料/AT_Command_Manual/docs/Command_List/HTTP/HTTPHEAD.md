## 查询HTTP头信息：AT+HTTPHEAD

语法规则：

| 命令类型 | 语法          | 返回                                      |
| -------- | ------------- | ----------------------------------------- |
| 执行命令 | AT+HTTPHEAD   | `+HTTPREAD:<date_len>`<br>`<data>` <br>OK |
| 测试命令 | AT+HTTPHEAD=? | OK                                        |

 

参数定义：

| 参数         | 定义             | 取值 | 对取值的说明 |
| ------------ | ---------------- | ---- | ------------ |
| `<date_len>` | 头信息的实际长度 |      |              |
| `<data>`     | 头信息           |      |              |
