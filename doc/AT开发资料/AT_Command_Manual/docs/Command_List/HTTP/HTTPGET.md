## 获取HTTP服务响应数据：AT+HTTPGET

语法规则：

| 命令类型 | 语法              | 返回                       |
| -------- | ----------------- | -------------------------- |
| 设置命令 | AT+HTTPGET=<path> | +HTTPGET:<date_len> <br>OK |

 

参数定义：

| 参数       | 定义             | 取值 | 对取值的说明                 |
| ---------- | ---------------- | ---- | ---------------------------- |
| <date_len> | 实际输出数据长度 |      |                              |
| <path>     | http下载文件路径 |      | 长度不超过255的ASCII字符串。 |
