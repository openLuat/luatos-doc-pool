## 初始化HTTP服务：AT+HTTPINIT

语法规则：

| 命令类型 | 语法                                                         | 返回 |
| -------- | ------------------------------------------------------------ | ---- |
| 执行命令 | AT+HTTPINIT                                                  | OK   |
| 测试命令 | AT+HTTPINIT=?                                                | OK   |
| 注意事项 | 1. 在使用HTTP服务前，应该先用AT+HTTPINIT命令来初始化HTTP协议栈<br>2. 如果使用的是https需要进行证书，加密套件，加密等级等相关的设置请参考SSLCFG指令 |      |
