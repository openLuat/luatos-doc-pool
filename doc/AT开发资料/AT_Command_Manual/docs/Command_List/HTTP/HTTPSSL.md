## 启用SSL：AT+HTTPSSL

语法规则：

| 命令类型 | 语法             | 返回                    |
| -------- | ---------------- | ----------------------- |
| 设置命令 | `AT+HTTPSSL=<n>` | OK                      |
| 读取命令 | AT+HTTPSSL?      | `+ HTTPSSL: <n>` <br>OK |
| 测试命令 | AT+HTTPSSL=?     | +HTTPSSL: (0-1)  <br>OK |

 

| 参数  | 定义             | 取值 | 对取值的说明 |
| ----- | ---------------- | ---- | ------------ |
| `<n>` | HTTP SSL功能开关 | 0    | 关闭SSL功能  |
|       |                  | 1    | 开启SSL功能  |
