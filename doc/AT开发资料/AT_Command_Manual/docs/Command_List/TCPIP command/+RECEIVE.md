## 多链接时接收数据：+RECEIVE

语法规则：

| 命令类型 | 语法                                 | 返回和说明                                                   |
| -------- | ------------------------------------ | ------------------------------------------------------------ |
| URC上报  | +RECEIVE,<n>,<length>: Received data | 注：Received data 是接收到的数据，和<br>+RECEIVE,<n>,<length>分开，另起一行 |

 

参数定义：

| 参数     | 定义                                  | 取值 | 对取值的说明         |
| -------- | ------------------------------------- | ---- | -------------------- |
| <n>      | Link No. <br>与+CIPSTRAT中<n>定义一致 | 0~5  | 整数型，表示连接序号 |
| <length> | 接收到的数据的长度                    | -    | 整数型               |
