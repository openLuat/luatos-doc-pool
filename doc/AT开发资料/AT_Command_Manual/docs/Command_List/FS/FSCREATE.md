## 创建一个文件：AT+FSCREATE

语法规则：

| 命令类型 | 语法                     | 返回              |
| -------- | ------------------------ | ----------------- |
| 设置命令 | `AT+FSCREATE=<filename>` | OK<br>或<br>ERROR |
| 查询命令 | AT+FSCREATE=?            | OK<br>或<br>ERROR |

 

参数定义：

| 参数         | 定义   | 取值 | 对取值的说明                           |
| ------------ | ------ | ---- | -------------------------------------- |
| `<filename>` | 文件名 |      | 字符串型，不用加双引号，不超过64个字节 |
