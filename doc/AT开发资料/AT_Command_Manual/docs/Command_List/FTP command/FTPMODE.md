## 设置FTP主动或被动模式：AT+FTPMODE

语法规则：

| 命令类型 | 语法               | 返回                    | 说明     |
| -------- | ------------------ | ----------------------- | -------- |
| 设置命令 | AT+FTPMODE=<value> | OK                      | 正常返回 |
| ERROR    | 输入格式有误       |                         |          |
| 查询命令 | AT+FTPMODE?        | +FTPMODE:<value> <br>OK |          |
| 测试命令 | AT+FTPMODE=?       | OK                      |          |

 

参数定义：

| 参数    | 定义            | 取值 | 对取值的说明         |
| ------- | --------------- | ---- | -------------------- |
| <value> | FTP主动被动模式 | 0    | 主动模式（暂不支持） |
|         |                 | 1    | 被动模式             |
