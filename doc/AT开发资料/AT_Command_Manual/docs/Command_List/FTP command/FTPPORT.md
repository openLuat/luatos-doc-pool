## 设置FTP控制端口：AT+FTPPORT

 

语法规则：

| 命令类型 | 语法               | 返回                    | 说明     |
| -------- | ------------------ | ----------------------- | -------- |
| 设置命令 | AT+FTPPORT=<value> | OK                      | 正常返回 |
| ERROR    | 输入格式有误       |                         |          |
| 查询命令 | AT+FTPPORT?        | +FTPPORT:<value> <br>OK |          |
| 测试命令 | AT+FTPPORT=?       | OK                      |          |

 

参数定义：

| 参数    | 定义          | 取值    | 对取值的说明 |
| ------- | ------------- | ------- | ------------ |
| <value> | FTP控制端口号 | 1~65535 | 缺省值为21   |
