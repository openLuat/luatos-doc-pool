## 设置FTP下载文件名称：AT+FTPGETNAME

设置服务器端目标文件的名称。

语法规则：

| 命令类型 | 语法                 | 返回                      | 说明         |
| -------- | -------------------- | ------------------------- | ------------ |
| 设置命令 | AT+FTPGETNAME=<name> | OK                        | 正常返回     |
|          |                      | ERROR                     | 输入格式有误 |
| 查询命令 | AT+FTPGETNAME?       | +FTPGETNAME:<name> <br>OK |              |
| 测试命令 | AT+FTPGETNAME=?      | OK                        |              |

 

参数定义：

| 参数   | 定义            | 取值 | 对取值的说明                  |
| ------ | --------------- | ---- | ----------------------------- |
| <name> | FTP下载文件名称 |      | 为长度不超过99的ASCII字符串。 |
